## :zero: About 
아티스트 및 행사를 관리하는 MVVM 기반 Pilot Proeject 

|Title|Contents|
|------|---|
|Notice|공지사항에 관련된 데이터 CRUD|
|Artist|아티스트에 관련된 데이터 CRUD|
|Event|행사에 관련된 데이터 CRUD|
